# San Diego Trip Guide: Ada's 6th Birthday 🎂

Welcome to the interactive guide for Ada's 6th birthday adventure in San Diego! This project is a multi-page, visually rich itinerary designed to make the trip as smooth and fun as possible.

## 🚀 Live Site
Check out the live guide here: **[https://scotthernandez.github.io/Ada6th/](https://scotthernandez.github.io/Ada6th/)**

## 🗓️ Trip Overview
- **Dates:** March 6-8, 2026
- **Basecamp:** Hyatt Vacation Club at The Welk (Escondido, CA)
- **Key Highlights:** Safari Park, Whale Watching, La Jolla Tide Pools, and Gourmet Sushi nights.

## ✨ Features
- **Interactive Itinerary:** Expandable day cards with checkable tasks.
- **Detailed Sub-pages:** Dedicated guides for Hotel, Safari Park, Coastal activities, and Dining.
- **Visual Richness:** AI-generated images and custom thumbnails for a premium feel.
- **Pro-Tips:** Kid-friendly advice tailored for a 6-year-old.

## 🛠️ Tech Stack
- **Structure:** HTML5
- **Styling:** CSS3 (Vanilla) with Glassmorphism and Micro-animations
- **Logic:** Vanilla JavaScript
- **Deployment:** GitHub Pages
